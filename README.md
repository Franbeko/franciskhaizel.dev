# Francis Kojo Haizel – Portfolio

Welcome to the source code of my personal developer portfolio.  
The site is built to be clean, fast, and fully responsive, with a strong focus on modern frontend practices and user experience.

🔗 **Live site:** [franciskhaizel-dev.vercel.app](https://franciskhaizel-dev.vercel.app/)

---

## 🧑‍💻 About the project

This portfolio showcases my work as a fullstack developer, highlighting key projects, technical skills, and my approach to building web applications.

It includes:
- Dark mode (default) with a toggle on larger screens
- Typing animations and a starfield background
- Fully responsive layout
- A working contact form (powered by Formspree)
- Smooth scroll navigation

## 🚀 Getting started (for developers)

If you want to run this project locally:

```bash
git clone https://github.com/Franbeko/franciskhaizel.dev.git
cd francis-portfolio
npm install
npm run dev

The site will be available at http://localhost:5173.

To create a production build:
npm run build

📁 Project structure (simplified)
src/
├── components/     # Hero, About, Skills, Projects, Contact, etc.
├── hooks/          # Custom hooks (toast, etc.)
├── lib/            # Utility functions
